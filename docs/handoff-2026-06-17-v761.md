### Root Cause: Commission Card ไม่โหลด / โหลดช้า (07b_nrr_target.js)

**`_dataKey()` ใน `_commGatedRender` ขาด `_tgtLoadedFromDB` flag:**

Flow ที่ผิด:
1. R2 data โหลดเสร็จ → `_commGatedRender()` ยิง
2. Supabase ยังไม่มา → `_tiersReady=false` → strip แสดง "กำลังโหลด"
3. key ถูก lock (`_lastCommKey = key`)
4. Supabase มา → `_tgtLoadedFromDB=true` → แต่ key เหมือนเดิม → skip render
5. commission card ค้าง shimmer ถาวร

### Fix (v761) — 07b_nrr_target.js

**Patch 1: `_dataKey()` เพิ่ม DB flag**
```js
var db = window._tgtLoadedFromDB ? '1' : '0';
return r + ':' + ph + ':' + hh + ':' + db;
```

**Patch 2: หลัง `loadTargets()` เสร็จ — ยิง gated render ชัดเจน**
```js
try{ if(typeof window._commResetKey==='function') window._commResetKey(); }catch(e){}
try{ if(typeof window._commGatedRender==='function') window._commGatedRender(); }catch(e){}
```
