Design Decisions

Hero Card — Runrate vs Target

ตัวเลขใหญ่ที่สุดคือ runrate ในมือจริง ไม่ใช่ยอดเดือนที่แล้ว Sales ต้องรู้ทันทีว่าตัวเองอยู่ที่ไหน

3-Month Projection Strip

แสดง M+0/M+1/M+2 พร้อม gap ให้ Sales รู้ล่วงหน้าว่าเดือนไหนจะขาด → ต้องปิดร้านเพิ่มตอนนี้

Handover Warning Card

แสดงเฉพาะร้านที่จะออกเดือนนี้ พร้อมบอก GMV ที่จะหาย ให้ Sales รู้ว่าต้องปิดใหม่เพิ่มเท่าไร

Days Held Progress Bar

แทน pace % ของ KAM ด้วย tenure bar — สีเขียว→เหลือง→แดง ตาม % ของวันที่ผ่านไป ไม่มี baseline concept

Pipeline — Manual Estimate

3 fields เท่านั้น: ชื่อร้าน + ยอดคาด + วันเริ่ม grouped by month พร้อม gap bar แสดง shortfall

Account View — เหมือน KAM

กดร้านแล้วเข้าได้ครบ: profile, SKU, Echo, Save เหมือน KAM แต่ซ่อน NRR bar / upsell / pace signal แสดง days_held + deadline แทน
