## งานที่ทำใน session นี้ — QNRR Sheet Redesign

### Design decisions (ยืนยันกับ Bucci)
1. **GMV บน bar** = monthly total GMV (ไม่ใช่ core-only) — bar แสดง portfolio ทั้งหมด, NRR% อยู่ hero zone แยกกัน ไม่ conflict
2. **Transfer out** — ไม่ปรับออกจากฐาน เพราะฐาน = Mar commitment ต้น Q; แสดงเป็น callout card แทน เพื่อให้เห็น true impact
3. **Breakdown table** — outlet count header แสดงเฉพาะ core_nrr (active) + core_nrr_churn เท่านั้น format: `436+40`

### CSS — styles_qnrr.css v775
**Semantic color palette ใหม่:**
- core_nrr:    `rgba(74,222,128,.86)` — เขียว (retention)
- handover:    `rgba(96,165,250,.80)` — ฟ้า (neutral/info)
- expansion:   `rgba(52,211,153,.72)` — teal (growth distinct from core)
- new_sales:   `rgba(167,139,250,.72)` — purple (new)
- comeback:    `rgba(251,191,36,.72)` — amber (warm)
- transfer_in: `rgba(255,255,255,.19)` — white dim
- churn:       `rgba(248,113,113,.84)` — แดง
- transfer_out: ghost (dashed border only)

**Layout ใหม่:**
- `.qnrr-hero-row` — flex row: base block LEFT + NRR cluster RIGHT
- `.qnrr-nrr-cluster` — 3 slot NRR% with separators (ย้ายออกจาก base-strip เป็น hero zone)
- `.qnrr-bar-top-wrap` — 32px (ลดจาก 44px) แสดง GMV only ไม่มี NRR%
- `.qnrr-view-toggle` — toggle กราฟ/Breakdown
- `.qnrr-breakdown-wrap` + `.qnrr-bk-table` — breakdown table CSS
- `.qnrr-tout-card` — transfer out callout card (แทน tout-note เดิม)
- **ลบออก:** `.qnrr-base-strip`, `.qnrr-stats-row`, `.qnrr-drill-sublbl`, `.qnrr-bar-nrr-below`, `.qnrr-bar-nrr-overlay` (ทั้งหมด source ของ metric duplication)
