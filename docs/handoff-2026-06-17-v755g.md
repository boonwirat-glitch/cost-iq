## ปัญหาที่ยังไม่แก้ ❌

### 1. Outlet tooltip ไม่แสดง (ปัญหาหลัก)
**อาการ:** `bulkSkuOutletData accounts: 0` ทุกครั้งที่ app โหลด — ไม่มี `⌄` บน SKU name — tap ไม่มีผล

**สาเหตุที่สงสัย:**
- IndexedDB cache tab `bundle-sku-outlet-{key}` อาจเก็บ empty content จากครั้งที่ 404 ก่อนหน้า → ingest → rowCount=0 → ดูเหมือน ok แต่ data ว่าง
- v755g เพิ่ม rowCount=0 guard แล้ว แต่ยังไม่ได้ verify ว่าแก้ได้จริง

**Debug commands:**
```js
// เช็ค outlet status

// เช็ค IDB cache
_csvCacheGet('bundle-sku-outlet-duangruedee_bu_freshket_co').then(r=>console.log('cache:', r?.value?.slice(0,50)))

// Force fetch
const sk = _kamSafeKey(currentUserProfile.email)
_fetchKamFile({url:`${R2_BASE}/sense_sku_outlet_${sk}.csv`,type:'bulk-sku-outlet',tab:`bundle-sku-outlet-${sk}`})
  .then(ok=>console.log('ok:',ok,'accounts:',Object.keys(bulkSkuOutletData).length))
```

**ถ้า force fetch สำเร็จ (accounts: 25) แต่ auto ไม่ทำงาน** → ปัญหาอยู่ที่ retry trigger ใน `_fetchKamBundle`

**แนวทางแก้ที่ยังไม่ลอง:**
- Clear IDB ทั้งหมดสำหรับ tab `bundle-sku-outlet-*` ตอน app init ก่อน fetch ครั้งแรก (nuclear option แต่ clean)
- หรือ เพิ่ม version suffix ใน tab name เช่น `bundle-sku-outlet-v1-{key}` → cache miss ทุก user → fetch ใหม่
