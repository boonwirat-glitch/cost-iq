## Key Architecture Notes

- **Scroll root:** `window` (body scroll) — iOS native momentum scroll บน separate thread จาก JS main thread
- **Scroll-linked approach:** ทำงานบน iOS เพราะใช้ rAF passive แต่ layout reflow ทุก frame อาจ jank ถ้ามี expensive DOM operations อื่น
- **Duplicate function risk:** การ replace function block ใน `06_portview_teamview.js` ต้องระวัง end_marker ให้ครอบ range ถูกต้อง ไม่งั้นจะ prepend duplicate
- **build pipeline:** `python3 build.py vN` → bump SW → push src → push index.html via Git Tree API
