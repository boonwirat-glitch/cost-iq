### Fix 1: Account View จอขาว (v752a)

**Root Cause:**  
`styles_restaurant.css` กำหนด `background:#fff` ให้ `#rest-sheet-body` และ `.scr` inside restaurant sheet  
Sheet สูง `height:calc(100dvh - 75px)` ตลอด — ถ้า content น้อยก็เห็นพื้นขาวโล่งครึ่งหน้าจอ  
account ที่ content เยอะไม่เห็นปัญหาเพราะ content scroll เต็ม

**Fix — `src/styles_restaurant.css` (3 จุด):**
```css
/* BEFORE */
#rest-sheet-body { background:#fff }
body.restaurant-sheet #swipe-grp-a .scr { background:#fff }
body.kam-mode #rest-sheet-body { background:#fff }
body.kad-mode #rest-sheet-body { background:#fff }

/* AFTER */
#rest-sheet-body { background:var(--tk-bg-base) }
body.restaurant-sheet #swipe-grp-a .scr { background:var(--tk-bg-base) }
body.kam-mode #rest-sheet-body { background:var(--tk-bg-base) }
body.kad-mode #rest-sheet-body { background:var(--tk-bg-base) }
```
`--tk-bg-base = #0d1f3c` (KAM dark background) — ทำให้พื้นที่ว่างใน sheet เป็นสีเข้มเหมือน app

### Fix 2: SKU Movement โหลดช้า (v752b)

**Root Cause:**  
`portviewSelectAccount()` → `switchAccount()` → `refreshAll()` → `renderKamThisMonth()` ถูกเรียกก่อนที่ `sku_current.csv` จะโหลดเสร็จ  
เมื่อ `sku_current.csv` มาถึง (line 773 ใน `02_data_pipeline.js`) จะ signal `RenderBus` ซึ่งมี 800ms debounce + call `refreshAll()` ทั้งก้อน  
ช่องว่างระหว่าง "render ครั้งแรก" กับ "ข้อมูลมา" = 5–6 วินาทีที่ user รอเห็น SKU section ว่าง

**Fix — `src/02_data_pipeline.js` (line 773):**  
เพิ่ม fast targeted re-render ของ KAM overview โดยตรงก่อน RenderBus signal:

```javascript
// BEFORE
if(currentAccountId&&bulkSkuCurrentData[currentAccountId]){
  D.sku_current=bulkSkuCurrentData[currentAccountId];
  if(window.RenderBus)window.RenderBus.signal('sku_current');
}

// AFTER (v752b)
if(currentAccountId&&bulkSkuCurrentData[currentAccountId]){
  D.sku_current=bulkSkuCurrentData[currentAccountId];
  // fast targeted re-render — ถ้า scr-overview on + KAM mode
  try{
    var _scrOv=document.getElementById('scr-overview');
    if(_scrOv&&_scrOv.classList.contains('on')&&
       typeof isKAM!=='undefined'&&isKAM&&
       typeof renderKamThisMonth==='function'){
      setTimeout(function(){ try{ renderKamThisMonth(); }catch(e){} }, 0);
    }
  }catch(e){}
  if(window.RenderBus)window.RenderBus.signal('sku_current');
}
```

ผลลัพธ์: SKU Signals / สัญญาณบวก โผล่ทันทีที่ข้อมูลมาถึง ไม่ต้องรอ 800ms debounce
