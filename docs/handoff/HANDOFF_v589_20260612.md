## Monitor วัน Launch

รัน query นี้เย็น 13 มิ.ย.:

```sql
SELECT error_type, COUNT(*), MAX(created_at)
FROM app_errors
WHERE created_at > NOW() - INTERVAL '24 hours'
  AND error_type IN ('ci_analyze_fail','ci_audio_gap','ci_gemini_badjson')
GROUP BY error_type;
```

ถ้าเจอ `ci_analyze_fail` ดู `error_message` — ถ้าเป็น `location is not supported` = Cloudflare routing issue (รอหาย), ถ้าเป็นอื่น = แจ้งผม
